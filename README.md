# 🍔 Byte - Sistema de Gestión de Pedidos

## 📌 Descripción

Byte es una aplicación desarrollada en Python utilizando Flet, que permite gestionar pedidos en un entorno tipo restaurante. El sistema simula una carta digital donde el usuario puede seleccionar productos, generar pedidos y obtener una boleta en formato PDF.

Este proyecto corresponde a una implementación parcial (60%) enfocada en la gestión de pedidos y la interacción con el usuario.

---

## 🚀 Funcionalidades implementadas

### 🧾 Gestión de pedidos

* Visualización de productos desde base de datos
* Agregar productos al pedido
* Eliminar productos del pedido
* Cálculo automático del total

### 🧮 Generación de boleta

* Creación de boleta en formato PDF
* Detalle de productos seleccionados
* Cálculo del total final
* Apertura automática del archivo generado

### 🖥️ Interfaz de usuario

* Interfaz gráfica desarrollada con Flet
* Navegación mediante barra inferior (Pedidos, Stock, Estadísticas)
* Visualización tipo carta con imágenes de productos

---

## 🛠️ Tecnologías utilizadas

* Python
* Flet
* SQLite (base de datos)
* ReportLab (generación de PDF)

---

## ⚙️ Instalación y ejecución

1. Clonar el repositorio:

```
git clone https://github.com/kkitaaa/BYTE.git
```

2. Instalar dependencias:

```
pip install flet reportlab
```

3. Ejecutar la aplicación:

```
python main.py
```
